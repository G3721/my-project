## 📚 使用指南

### 基本功能
1. **选择章节**: 点击顶部4个按钮切换章节
2. **查看经文**: 每条经文包含：
   - 梵文原典（Devanagari字体）
   - 音译（IAST标准）
   - 发音（中文拼音）
   - 英文翻译
   - 中文释义

3. **学习发音**: 查看页面底部的发音指南

### 学习建议

#### 初学者
1. 先读中文和英文理解意思
2. 再看梵文原典
3. 学习音译和发音
4. 最后背诵梵文

#### 进阶者
1. 直接从梵文开始
2. 对照音标加深理解
3. 练习背诵梵文原典
4. 体会音韵之美

#### 每日练习
- 第1周：每天学习1-2条经文
- 第2-4周：复习并背诵
- 持续：加入冥想实践

## 🎵 添加音频功能

如果你有瑜伽经的音频文件：

1. 将MP3文件放入 `audio/` 文件夹
2. 命名格式：`章节-经文.mp3`
   - 例如：`1-1.1.mp3` 表示第1章第1条经文

3. 在 `js/app.js` 中添加音频标签
```javascript
// 在 createSutraCard 函数中添加：
card.innerHTML += `
    <audio controls>
        <source src="audio/${sutra.id}.mp3" type="audio/mpeg">
        您的浏览器不支持音频播放
    </audio>
`;
```

## 📝 自定义内容

### 修改颜色主题
编辑 `css/style.css` 第5-8行：
```css
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);
```
改为其他颜色组合，例如：
- 日出橙：`#f093fb 0%, #f5576c 100%`
- 森林绿：`#0ba360 0%, #3cba92 100%`
- 海洋蓝：`#2193b0 0%, #6dd5ed 100%`

### 添加更多经文
编辑 `data/yoga-sutras.json`：

```json
{
  "id": "1.xx",
  "sanskrit": "अध्यायः सूत्रम्।",
  "transliteration": "Adhyāyaḥ sūtram.",
  "pronunciation": "阿-德-希-亚-亚-赫 苏-特-拉-姆",
  "english": "Chapter, verse.",
  "chinese": "章节，经文。"
}
```

## 🆘 常见问题

**Q: 梵文显示不正常？**
A: 确保网络连接正常，页面需要加载Google Fonts

**Q: 如何在手机上使用？**
A: 网站支持响应式设计，直接用手机浏览器打开

**Q: 能否离线使用？**
A: 可以，需要先打开一次加载字体和脚本

**Q: 如何添加新章节？**
A: 修改 `data/yoga-sutras.json` 添加新章节结构
