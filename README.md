帕坦伽利《瑜伽经》完整三语对照学习网站 - 梵文、英文、中文

## 📖 项目简介

这是一个极简的瑜伽经学习网站，提供帕坦伽利《瑜伽经》全文193条经文的三语对照版本：

- **梵文原典** (Sanskrit) - 使用Devanagari字体展示
- **英文翻译** (English) - 准确的英文释义
- **中文翻译** (Chinese) - 详细的中文解释

## ✨ 特色功能

- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🎨 **极简设计** - 专注于阅读体验，无复杂交互
- 📄 **完整收录** - 包含全部4章193条经文
- 🖨️ **打印友好** - 优化的打印样式
- ⚡ **快速加载** - 静态网站，无需服务器
- 🔍 **SEO优化** - 搜索引擎友好

## 📚 内容结构

### 四个章节
1. **第1章：集中心量篇** (Samādhi Pāda) - 51条经文
2. **第2章：修行精进篇** (Sādhana Pāda) - 53条经文
3. **第3章：神奇成就篇** (Vibhūti Pāda) - 55条经文
4. **第4章：解脱自在篇** (Kaivalya Pāda) - 34条经文

### 每条经文包含
- **经文编号** - 传统的章节+经文编号
- **梵文原文** - Devanagari字体展示
- **IAST音译** - 梵文罗马音转写
- **英文翻译** - 准确的英文释义
- **中文翻译** - 详细的中文解释

## 🚀 本地运行

### 方法1：直接打开
```bash
# 直接用浏览器打开
open index.html
```

### 方法2：本地服务器
```bash
# 使用Python
python3 -m http.server 8000

# 使用Node.js
npx serve .

# 使用PHP
php -S localhost:8000
```

然后在浏览器访问 `http://localhost:8000`
